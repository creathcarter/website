Remote Mining Town - Mesarthim

Work Contributed:
Environment Design/Modeling
Texture Creation
Level Scripting


Polycount (triangles): 29,612 (does not include active particles)
Texturememory: 18MB
Models created in 3dsMax 5.1
Textures created in Photoshop 7 using a combination of hand painted work and digitally edited photographs.

The focus of this environment was to experiment with modular construction, while trying to match todays graphical standards for a PC RPG. This project lasted for roughly 2 weeks.
