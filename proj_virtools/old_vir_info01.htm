Small Interactive Environment created for the purpose of growing familiar with the Virtools Dev development application.
The playable environment showcased a third person avatar based navigation system, LOD modeling, and basic particle effects.
Virtools Dev is a software application that is used for graphical construction of engine, game and AI code, in place of a programming langage.

This project lasted for 3 days.
