Unreal Tournament Environments

Hohenheim - Mountain Lake Regions


Models created in 3dsMax 5.1
Textures created in Photoshop 7
Audio created in Sound Forge 4.5

This project was created for application to a specific company. Requirements were to create an environment suitable for a persistent world, no polycount / texture limit given and was designed with todays PCs in mind.
Recommended Specs: P3/4 2Ghz, 256MB RAM, GeForce 4, equivalent or greater.

Duration of this project is estimated to have lasted for 3 weekends (6 days), between 8 - 12 hours per day. All design, static meshes, and nearly all but a few textures were created specifically for this environment. Windmill/Mechanical audio effects created specifically for this environment by the author.
