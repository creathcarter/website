Mesarthim -Environment Walkthrough (23.2 MB)

This is a walkthrough of a small environment. No gameplay mechanics or other interactivity currently present.

This interactive environment plays in your web browser via the Virtools Webplayer. If this is your first time using the Virtools Webplayer it will automatically download and install itself as a plugin when you click the link. (very minimal file size)

Controls:

Movement - W, A, S & D keys

Viewpoint - Mouse

Switch to Fullscreen - Right click in the window and select "fullscreen". Hit ESC at any time to return to windowed mode.

Click Here to begin.

 

Warning: For some reason the Virtools Webplayer has a lighting issue with video cards using Texture and Lighting technology. This results in some light sources not displaying correctly in the environment. If you notice this issue, or have a video card that is using T&L, then right click in the game window, select settings, and change the top most drop down menu to "Primary Display Driver (Hardware DX7)".

This should enable the environment to display correctly.
