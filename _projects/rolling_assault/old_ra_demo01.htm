Rolling Assault -Interactive Demo (15.4 MB)

This is a single level of a downloadable webgame in late Alpha stage development.

This game is a simple side-scrolling action game. The player controls a tank with realistic physics as it travels over uneven terrain and destroys enemies with two available weapons.
The demo ends immediately prior to the first stage boss.

Controls:

Mouse - Controls the direction of the cannon.
Left Mouse Button - Fire Cannon
Right Mouse Button - Fire Rockets
Left / Right Arrow Keys - Move Tank Forward / Backward
Esc - Quit game
