Muridae Massacre

Work Contributed:

Texture Creation (Environments)
Particle/Effects Creation
Co-Game Design
GUI Design

The sequel/update to "When Lemmings Attack" (see Projects section). This project was created with a team of individuals for the purpose of entering the Independant Game Festival.

As a parody of the game "Lemmings", the purpose of this title was to plant traps in the paths of the Lemmings in attempt to stop them from reaching the level exit. Each set of stages had a set "theme" for the Lemming appearance and surrounding environment, with the featured theme of the final demo being the Medieval Era.
This project was completed using Virtools Dev, which is a software application that is used for graphical construction of engine, game and AI code, in place of a programming langage.

The project cycle lasted from April 2002 to September 2002.
